#### Scikit Learn 數字辨識

1. Load data
2. Set a classifier
3. Laern a model
4. Predict the result
5. Evaluate

> scikit learn 有很多 classifier，只需要參考 doc 給他參數就能夠使用了

#### 前處理（對原始資料先做些處理）

1. Clean data
2. Feature extraction
3. Convert category and string to number
4. Sparse data
5. Feature selection

> raw data -> [前處理] -> 餵進去的向量
